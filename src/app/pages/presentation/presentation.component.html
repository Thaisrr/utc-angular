Présentation

Installation

Créer un projet Angular :

ng new mon-projet

Créer un composant

ng generate component repertoire/mon-composant ( --skip-tests )

ng g c ....

Un composant est composé de 3 fichiers :

  • Logique : .ts -> classe typescript
  • Template : .html -> HTML amélioré
  • Style : .css -> style associé au composant, valable uniquement pour le composant

Interpolation

Interpréter des expressions JS dans le template

{{message}}

1 + 1 = {{1 + 1}}

{{message.toUpperCase()}}

{{displayText()}}

Rien : {{displayVoid()}}

Dans les balises moustaches, on peut afficher n'importe quelle expression JS, tant qu'elle retourne quelque chose.
